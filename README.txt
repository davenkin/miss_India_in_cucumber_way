Before run, please make sure you have Ruby 1.9.2 installed, other versions may
also work(I haven't try yet) by altering the relevant sections in .rvmrc file.


First time to run the tests, please follow the steps below to run all the cucumber test
     (1)cd path/to/the/cucumber/project
     (2)./run
Afterwards,once all the gems have already been installed, you may use the follow commands
to run the tests:
     (1)cd path/to/the/cucumber/project
     (2)cucumber

Every Scenario will open a new browser window, so if you don't want to run all the tests
in one time, please use tag to run a specific Scenario.
